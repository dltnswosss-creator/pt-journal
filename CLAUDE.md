# PT Journal 프로젝트 개요

네이버 밴드에 회원들의 운동일지를 효율적으로 작성하기 위한 웹 앱.

## 핵심 흐름

1. 튜터(트레이너)가 자신의 계정으로 로그인
2. 회원별 운동 기록 입력: 운동 이름, 중량, 세트 수, 반복 횟수
3. 각 운동에 대한 피드백 + 전체 운동 총 피드백 작성
4. 완성된 일지를 네이버 밴드에 바로 복사·붙여넣기 가능한 형식으로 출력

## 목적

튜터가 매번 밴드에 직접 타이핑하는 반복 작업을 줄이고, 일지 양식을 통일하기 위함.

## 작업 시 주의사항

- 네이버 밴드 복사용 출력 형식을 항상 유지할 것
- 튜터별 로그인 구조를 고려할 것
